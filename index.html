What We Do

Integrated Circuits (ASIC & FPGA) described in VHDL & (System)Verilog; verified and synthesized with open source tools; and printed with open standard cells.

01

Describing ICs

Integrated Circuits modeled in System Description Languages such as SystemC or SystemVerilog and described in traditional Hardware Description Languages such as VHDL or Verilog

02

Verifying ICs

Integrated Circuits simulated (System Level, RTL and Switch Level) and verified (RTL and Switch Level) with open source tools such as GHDL (VHDL), Icarus Verilog (Verilog) or Verilator (Verilog-C/C++)

03

Synthesizing ICs

Integrated Circuits optimized and synthesized with open source tools such as Yosys-ABC (Verilog)

04

Printing ICs

Integrated Circuits placed, routed and printed with open source tools such as Magic VLSI and using open standard cells

Printed Circuits Boards drawed, printed, verified and tested with open source tools.

01

Drawing PCBs

Printed Circuit Board Schematic Diagrams drawed with open source tools

02

Printing PCBs

Printed Circuit Board Mask Layouts printed with open source tools

03

Verifying PCBs

Printed Circuit Board Mask Layouts verified with open source tools

04

Testing PCBs

Printed Circuit Board and Integrated Circuit Mask Layouts tested with open source tools
